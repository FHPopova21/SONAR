## 🎯 Въведение и Мотивация
В съвременната градска среда, масовото използване на слушалки с активно шумоподтискане (ANC), както и предизвикателствата пред хората с частична или пълна загуба на слуха (над 400 млн. души глобално), създават сериозни рискове за безопасността. 

**SONAR** решава този проблем, като действа като "дигитален слух". Системата непрекъснато сканира акустичната среда, разпознава критични звукови събития (сирени на линейки, полицейски автомобили, клаксони, счупване на стъкло) и мигновено уведомява потребителя чрез визуални и хаптични (вибрационни) сигнали.

## 🏗 Архитектура на системата

Проектът е изграден като хибридна, крос-платформена екосистема, която гарантира 100% поверителност на данните, тъй като всички изчисления се извършват **локално на устройството (On-device / Edge Computing)**, без изпращане на аудио към облачни сървъри.

* **iOS & watchOS Клиент:** Нативни приложения, разработени със Swift и SwiftUI. Използват `Core ML` за изпълнение на невронната мрежа директно върху Neural Engine-а на Apple устройствата.
* **Windows Desktop Клиент:** Хибридно приложение, съчетаващо модерен потребителски интерфейс на React с мощен Python бекенд. Двете среди комуникират в реално време чрез WebSockets (чрез библиотеката `Eel`).

## 🧠 Изкуствен интелект и Машинно обучение

Ядрото на SONAR е дълбока конволюционна невронна мрежа (CNN), обучена да класифицира аудио сигнали с изключителна скорост.

<details>
<summary><b>📊 1. Данни и Препроцесинг (Кликнете за детайли)</b></summary>
<br>
Аудио сигналите не се подават директно на модела. Вместо това те се трансформират във визуална репрезентация:

* **Библиотека:** `librosa` (Python).
* **Екстракция на признаци:** Аудиото се конвертира в **Mel-Spectrograms** (Мел-спектрограми). Това превръща задачата за разпознаване на звук в задача за компютърно зрение (Computer Vision), което позволява използването на високоефективни CNN архитектури.
</details>

<details>
<summary><b>⚙️ 2. Модел и Обучение (MobileNetV2)</b></summary>
<br>

* **Архитектура:** Използвана е `MobileNetV2` – архитектура, специално оптимизирана за мобилни устройства и хардуер с ограничени ресурси. Тя предлага перфектен баланс между висока точност и нисък разход на батерия/памет.
* **Фреймворк:** Моделът е обучен с **PyTorch**.
* **Оптимизация:** След обучението, моделът е конвертиран в `.mlmodel` формат за интеграция в екосистемата на Apple, като е приложена квантизация (Quantization) за допълнително намаляване на размера му и ускоряване на инференцията до под 100 милисекунди.
</details>

## 🚀 Инсталация и локално стартиране

### 🖥️ За Windows (Desktop Приложение)

**Вариант 1: За крайни потребители (Препоръчително)**
1. Изтеглете файла `SONAR_Setup.exe` от секцията [Releases](../../releases).
2. Инсталирайте и стартирайте програмата.

**Вариант 2: За разработчици (От сорс код)**
```bash
# 1. Клонирайте хранилището
git clone https://github.com/FHPopova21/Ambient-Sound-Awareness-System-ASAS-.git

# 2. Инсталирайте Python зависимостите
cd Windows_App/backend
pip install -r requirements.txt

# 3. Инсталирайте React зависимостите и генерирайте UI
cd ../frontend
npm install
npm run build

# 4. Стартирайте приложението
cd ../backend
python main.py

```

### 🍎 За iOS / watchOS

1. Отворете `Mobile_App/SONAR.xcodeproj` с **Xcode 15+**.
2. В секцията *Signing & Capabilities*, задайте вашия Apple Developer Team.
3. Уверете се, че файлът `MobileNetV2.mlmodel` е наличен в дървото на проекта.
4. Изберете целево устройство (физически iPhone или симулатор) и натиснете `Cmd + R`.
